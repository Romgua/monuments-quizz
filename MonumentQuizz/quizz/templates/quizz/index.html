Monuments Quizz

Règles du jeu

L'application Monuments Quizz est un site web qui a pour objectif de tester les connaissances de l'utilisateur sur les monuments mondiaux et leurs villes.
Le système de jeu se base sur un système de point qui récompense la prise de risque. Sur une série de 10 questions, l'utilisateur sera amené à choisir, pour chaque question, un mode de réponse afin d'identifier le lieu du monument visualiser à l’écran.

Les différents modes de choix et le nombre de points récompensés sont décrits ci-dessous :

Modes Points Propositions
Duo +1 point 2 suggestions
Carré +3 points 4 suggestions
Cash +5 points Aucune suggestion
